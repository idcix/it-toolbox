# Changelog - v2.0.0

**发布日期**: 2026-03-24

IT Toolbox v2.0.0 从 v1.0.0 的 87 个工具扩展到 96 款工具，新增 9 个实用工具。

---

## 新增工具（9个）

### 汇率换算
- 实时汇率换算，支持 150+ 货币
- 热门汇率展示
- KV 缓存加速（1 小时）
- 使用 exchangerate-api.com 数据源

### HTTP API 测试
- 类 Postman 界面
- 支持 GET/POST/PUT/DELETE/PATCH/HEAD/OPTIONS 方法
- 自定义 Headers 和 Body
- 响应状态码、时间、大小展示
- 后端代理解决跨域问题

### 二维码识别
- 上传图片识别二维码
- 摄像头实时扫码
- 支持拖拽上传
- 使用 jsqr 前端解析

### AI 代码 Review
- AI 进行 Code Review
- 安全漏洞、性能、可读性三维分析
- 评分可视化展示
- 问题与建议分离展示

### JSONPath 查询
- 使用 JSONPath 表达式查询 JSON 数据
- 支持复杂过滤表达式
- 常用表达式快捷选择
- 语法参考提示

### JSON 转 CSV
- 将 JSON 数组转换为 CSV 格式
- 支持自定义分隔符（逗号/分号/Tab/竖线）
- 可选是否包含表头
- 支持下载 CSV 文件

### 数学表达式计算
- 计算复杂数学表达式
- 支持函数（sqrt、pow、sin、cos、log 等）
- 支持常量（pi、e、phi）
- 支持单位换算（inch to cm 等）
- 使用 mathjs 库

### HTML 实时预览
- 实时预览 HTML/CSS/JS 代码
- 支持自动刷新
- 支持全屏预览
- 支持在新标签页打开
- iframe sandbox 安全隔离

### Meta 标签生成
- 生成 SEO 友好的 Meta 标签
- 支持 Open Graph 标签
- 支持 Twitter Card 标签
- 支持页面类型选择（网站/文章/产品）
- 一键复制生成的标签
